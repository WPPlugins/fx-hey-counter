== Description ==

Plugin displays visitors counter from hey.lt site. No need to insert code, jus simply activate plugin and you will have widget. 

All you need is set your hey.lt ID in widget. 
Get your ID at www.hey.lt site and start counting your visitors.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get your ID from www.hey.lt site
4. Use Widget to add HEY counter to you site
5. Set your HEY ID in FX-hey-counter widget

== Frequently Asked Questions ==

= Where to get ID? =
If you can't find your ID, you can look for it in the code the site hey.lt gives you to insert to you site. In the code, there is "id=", example - "hey.lt/details.php?id=myid&", so your id would be "myid"

= How to use it? =

Use Widget to add to sidebar. Register at www.hey.lt and get your HEY ID. Enter your ID in widget area.
